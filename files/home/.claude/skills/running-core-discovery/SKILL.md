# Running a CORE Discovery session

Facilitate a collaborative session that aligns a client on three pillars and produces a CORE Brief. You are the facilitator, not an order-taker: build the output in steps with the room participating, so they own the result.

The three pillars: **Brand** (who you are) × **User** (their needs) × **Goals** (the business). Every exercise feeds one of these.

## Principles of alignment (the mechanism — apply throughout)

1. **Safe container** — set three rules aloud: Participate, Say Anything, Have Fun. Redirect every negative to its positive form ("disorganized" → "organized").
2. **Inclusion** — excluded stakeholders sabotage the output with hidden agendas. Get everyone heard.
3. **Prioritize together** — always end an exercise with a forced ranking (Top 5 → Top 1). Un-prioritized output produces burnout and no decision.

## Pre-session checklist

- Send agenda, goals, and attendee list a few days ahead. Confirm group size ≤ 7.
- Privately brief the CEO/primary stakeholder: explain the flow, get buy-in, ask them not to veto team suggestions.
- Set up the board (physical or Mural/Miro). Prepare large sticky pads, markers, a timer.

## Session sequence

Run these in order. Time-box each and announce the timer. Use progressive passes (brainstorm all → Top 5 → Top 1), not column-by-column. Break every ~90 minutes.

1. **Team Alignment (open).** Each person names 3 things they want from today (model it yourself first). Then: "If you could get only ONE, what is it?" Write the three rules on the board.
2. **Brand Attributes.** Six columns of single-word adjectives — Culture, Customers, Voice, Feeling, Impact, X-Factor. Progressive passes to Top 1 each. Plug the Top 1s into the Mad-Lib brand statement. Use *ambiguation*: don't reveal the final statement until the end so no one anchors.
3. **Customer Profiles.** Build 3 profiles (e.g. Current, Ideal, plus one of Employees/Influencers). Four quadrants each: Demographics, Story, Needs (the need itself, not a feature), Solutions (go big). A value proposition = the need stated from the brand's POV.
4. **Goals Prioritization (Stack Ranking).** List goals across Revenue, Awareness, Efficiency. Score each on Desirability (1–10) + Doability (1–10) and assign a term (short/medium/long). Decision rule: Desirability + Doability ≥ 15 and short-term = do it now.
5. **CORE Positioning** *(secondary)*. Narrow Brand Attributes to Voice / Feeling / Impact; vote; form a short narrative phrase and 3 design directions.
6. **Feature Brainstorm** *(secondary)*. 3×3 grid of Top customer needs × Top goals; for each cell ask "what idea serves both?" Find win-wins.
7. **Customer Journeys** *(secondary, the most important of the three)*. Map each profile through Discover → Engage → Act → Return → Recommend → Advocate, using Feature Brainstorm ideas as the backlog.

## Close & deliver

- Confirm everyone got their "one thing." State next steps: "We'll summarize and send for review and approval."
- Translate alignment into design via **Anchor → Synthesize → Visualize**.
- Compile outputs into a **CORE Brief** and get explicit client approval before any design begins.
